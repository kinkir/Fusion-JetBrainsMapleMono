基于**JetBrains Maple Mono**([JetBrains Maple Mono](https://github.com/SpaceTimee/Fusion-JetBrainsMapleMono))构建纯英文字体

* 适用平台: Any

发布文件按 JetBrainsMapleMono-[NF/XX]-[NL/XX]-[HT/XX].zip 的格式命名:

1. XX: 占位符，表示该字体没有增加这一特性
2. NF: Nerd Font，为部分开发工具、命令行终端、代码编辑器等提供图标支持 (会导致字体文件体积略微增大)
3. NL: No Ligatures，禁用连字
4. HT: Hinted，使字体在低分辨率屏幕上 (<=1080P) 的渲染更加均匀 (可能会导致字体在高分辨率屏幕上的渲染略微模糊)
